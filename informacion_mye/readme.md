# Información mútua y entropía

## ¿Qué necesitas para usarlo?

El software que necesitas para poder ejecutarlo es [Go](https://go.dev/)

Sígue las instrucciones de la página para poder instalarlo en tu sistema operativo de preferencia, este programa específicamente no está limitado a el uso de un sistema operativo.

Para comprobar que lo tienes instalado ejecuta el siguiente comando `go version` en la terminal

## ¿Cómo puedes ejecutarlo?

Para ejecutar el programa dirijase al directorio del projecto, en una terminal ejecute el siguiete comando `go run .`. Si quiere generar el compilado ejecute el siguiente comando `go build -o informacion_mye.exe main.go`.
